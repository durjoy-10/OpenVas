# 🛡️ OpenVAS (GVM) Installation & Usage Guide (Kali Linux)

---

## Install OpenVAS (GVM):
```
sudo apt install gvm -y

```

## Setup OpenVAS (First Time Only) 
```
sudo gvm-setup

```

### ✅ After Setup:
#### ==> A username and password will be generated
#### ==> Save these credentials for login

## Check Installation Status:
```
gvm-check-setup

```

## Start OpenVAS
```
sudo gvm-start

```
### 🌐 Access Web Interface:
```
https://127.0.0.1:9392

```

## Stop OpenVAS:
```
sudo gvm-stop

```

## Restart OpenVAS:
```
sudo gvm-stop
sudo gvm-start

```
